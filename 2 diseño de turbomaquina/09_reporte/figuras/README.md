# Figuras del informe

Recursos gráficos usados por `Informe_Diseno_Ventilador_Sirocco.pdf`.

- Las gráficas numéricas se generan con Matplotlib desde los resultados Python.
- Las capturas 3D provienen de `10_interfaz_3d/`.
- La geometría 2D proviene del SVG de `07_planos/`.

No cambiar una figura sin verificar también el texto y los valores del informe.
Las imágenes pueden regenerarse ejecutando
`python3 08_software/generar_reporte_pdf.py`; las capturas 3D se actualizan al
modificar la interfaz o el modelo.
