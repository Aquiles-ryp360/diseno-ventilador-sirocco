# Examen_Turbomaquinas_Resolucion_Completa

Resolución universitaria formal de un examen de Turbomáquinas, con desarrollo matemático,
fundamento físico, hipótesis, verificación dimensional, esquemas y resultados en unidades SI.

## Compilación

```bash
cd latex && pdflatex resolucion_examen_turbomaquinas.tex
```

Para actualizar correctamente el índice, conviene ejecutar `pdflatex` dos veces. El flujo de
compilación del proyecto copia el PDF definitivo a:

```text
pdf/resolucion_examen_turbomaquinas.pdf
```

## Advertencia sobre las figuras

Las figuras 1 a 5 citadas en varios enunciados no fueron proporcionadas. En esos casos el
documento conserva expresiones paramétricas o desarrolla el criterio físico y el esquema
conceptual pertinente, sin inventar geometrías ni datos numéricos.
