# Manual consolidado del proyecto

Esta carpeta reúne en un solo documento la explicación de todas las carpetas
del proyecto.

- `Manual_Documentacion_Proyecto_Sirocco.pdf`: versión para leer o imprimir.
- `Manual_Documentacion_Proyecto_Sirocco.docx`: versión editable en Word.
- `Manual_Documentacion_Proyecto_Sirocco.html`: versión intermedia navegable.

Los documentos individuales permanecen dentro de cada carpeta bajo el nombre
`DOCUMENTACION.md` o `README.md`. En caso de cambios técnicos, actualizar
primero el documento individual correspondiente y volver a generar este manual.

```bash
python3 08_software/generar_manual_documentacion.py
```
