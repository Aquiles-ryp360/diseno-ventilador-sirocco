# Recursos de la presentación

Esta carpeta almacena imágenes utilizadas por la presentación web y el
generador PowerPoint.

`sirocco_fan_render.jpg` debe mantenerse con el mismo nombre mientras
`generate_pptx.py` lo referencie como `assets/sirocco_fan_render.jpg`.

`placeholder.txt` solo conserva la carpeta cuando no existen otros recursos y
puede mantenerse sin modificar.
